# Claude CLI Integration

## Installation
- Claude CLI is available via npm:
```sh
npm install -g claude-cli
```
- Or see: https://github.com/anthropic-ai/claude-cli

## Authentication
- Set your API key (get from Anthropic dashboard):
```sh
claude config set api_key <YOUR_API_KEY>
```

## Usage
- Chat interactively:
```sh
claude chat
```
- Send a prompt:
```sh
claude prompt "Summarize this text..."
```

## Project Usage
- Use for AI-powered summarization, Q&A, or code generation tasks.
