# Codex CLI Integration

## Installation
- Codex CLI is available via npm:
```sh
npm install -g codex-cli
```
- Or see: https://github.com/openai/codex-cli

## Authentication
- Set your OpenAI API key:
```sh
codex config set api_key <YOUR_API_KEY>
```

## Usage
- Run code generation:
```sh
codex generate "Create a React component for a login form"
```
- Chat interactively:
```sh
codex chat
```

## Project Usage
- Use for code generation, refactoring, or AI-powered CLI tasks.
